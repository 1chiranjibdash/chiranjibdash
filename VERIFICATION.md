## 📞 Support

### If Something Isn't Working

1. **Check console errors** (F12 → Console)
2. **Verify file paths** (relative, not absolute)
3. **Clear browser cache** (Ctrl+Shift+R)
4. **Test on different browser**
5. **Test on mobile** (Chrome DevTools)

### Common Issues

| Issue | Solution |
|-------|----------|
| Styles not loading | Check CSS file path in HTML |
| Navigation not working | Verify file names and extensions |
| Mobile view broken | Check responsive.css is linked |
| Form not submitting | Check browser console for errors |
| Images not showing | Verify image paths are correct |
