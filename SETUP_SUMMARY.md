## 🚀 How to Test Locally

### Option 1: Direct File Opening
```bash
# Navigate to project folder
cd portfolio

# Open index.html in your browser
# File > Open File > src/index.html
```

### Option 2: Using Python Server
```bash
cd portfolio
python -m http.server 8000
# Visit: http://localhost:8000/src/
```

### Option 3: Using Live Server (if installed)
```bash
cd portfolio
npm start
# or
npm run serve
```

## 📞 Support & Customization

### Common Customizations

1. **Change Primary Color**
   - Edit `--highlight: #e94560` in `css/style.css`
   - This affects all accent colors

2. **Add New Project**
   - Copy a project card in `projects.html`
   - Update title, description, tags, and links

3. **Update Skills**
   - Edit skill items in `about.html`
   - Add/remove categories as needed

4. **Change Font**
   - Update `font-family` in `css/style.css`
   - Or add custom @font-face
