⚡ QUICK CUSTOMIZATION TIPS

Change Colors:
→ Open css/style.css
→ Find :root section
→ Change --highlight: #e94560 to your color
→ Save and refresh browser

Change Fonts:
→ Open css/style.css
→ Find body { font-family: ... }
→ Replace with your font choice

Adjust Spacing:
→ Find .container { padding: 0 2rem }
→ Adjust the numbers for more/less space

Update Content:
→ Edit HTML files directly
→ Find the text you want to change
→ Replace with your own text
→ Save and refresh
